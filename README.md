# Projet NEO
Projet de gamification du recrutement.

# Directory structure

Project directory
``` bash
├───node_modules
├───public
│   ├───assets
│   │   ├───css
│   │   ├───fonts
│   │   ├───js
│   │   └───sass
│   │       ├───base
│   │       ├───components
│   │       ├───layout
│   │       └───libs
│   └───images
├───routes
└───views
    ├───pages
```

# Installation
```
git clone https://github.com/ClementWishow/Neo.git
npm install
node server 
```
