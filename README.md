# Projet NEO
Projet de gamification du recrutement.

# Directory structure

Project directory
``` bash
├───app
│   ├───configuration
│   ├───controllers
│   ├───models
│   ├───queries
│   ├───routes
│   │   ├───index
├───node_modules
├───public
│   ├───assets
│   │   ├───css
│   │   ├───fonts
│   │   ├───js
│   │   └───sass
│   │       ├───base
│   │       ├───components
│   │       ├───layout
│   │       └───libs
│   └───images
└───views
    ├───pages
```

# Installation
```
:information_source: node >= 16

git clone https://github.com/ClementWishow/Neo.git
npm install
npm start 
```
